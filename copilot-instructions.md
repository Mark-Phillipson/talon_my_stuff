# Copilot Instructions

For all terminal commands requested by the user:
- Always return command lines inside markdown code blocks (````bash` + code + ```` or ````powershell` + code + ````) so they can be copied and pasted directly.
- Include any necessary context/steps before or after the code block in normal text, but ensure command lines are clearly delimited.

Example:

```bash
adb devices -l
adb connect 192.168.1.42:5555
```
