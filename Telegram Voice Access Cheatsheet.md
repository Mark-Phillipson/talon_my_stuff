Telegram Voice Access Cheat Sheet:

1. Open Telegram chat.
2. Say "show keyboard" or "tap keyboard" to bring up the on-screen keyboard.
3. Say "tap microphone" (or "tap [number]" if numbers are shown) to activate voice typing.
4. Dictate your message.
5. Say "tap send" or "tap [number]" (matching the send button) to send the message.

Tips:
- If a command doesn't work, try using the number overlay ("show numbers") and then "tap [number]".
- For editing, say "select [text]" or "move cursor [direction]" as needed.
- You can repeat steps 2–5 for each new message.
